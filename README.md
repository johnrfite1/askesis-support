## Frequently Asked Questions

**Q: How do I change the sprint duration?**
A: Tap the (+) or (-) buttons in the "Session Setup" panel on the main screen. You can adjust Sprint, Recovery, and Rounds independently.

**Q: Does the timer work in the background?**
A: Yes. Askesis I/O uses native local notifications to alert you when a round is complete, even if the phone is locked or you are using another app.

**Q: Can I use this without internet?**
A: Yes. Askesis I/O is 100% offline and requires no network connection.
